Variables e Info, desde la carga de contenido

Invocación de variables

Las variables, al cargar contenido desde esta sección, se deben invocar con doble llaves de apertura y de cierre, envolviendo el nombre de la misma.
Por ejemplo: {{nombreVariable1}}.

Otro ejemplo: {{nombreVariable2}}.

Gestión de variables:

Sus valores, se deben de administrar desde una página específica:  'Template Variables'

Capturar info mediante Deki

Además, se pueden usar en conjunto con Deki, por ejemplo, para invocar el nombre del usuario que esté accediendo:

Hola {{ User.Name }}, bienvenido, que tengas un excelente {{ Date.DayName(Date.Now) }}.

Donde "User.Name" es el nombre del usuario, y "Date.DayName(Date.Now)", es una función default de Nice, mediante la cuál podemos capturar que día es.

Condicionales ternarios desde la carga de contenido

También pueden usar condicionales desde este sector, con la siguiente sintaxis:

Hola {{ User.Anonymous ? 'visitante anónimo' : User.Name }} !

La cual significa, que, si el usuario, efectivamente fuera anónimo, veremos "Hola visitante anónimo", por el contrario, si fuera un usuario registrado: "Hola Juan"
